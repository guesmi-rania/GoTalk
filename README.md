GoTalk

GoTalk est une application de chat en temps réel construite avec Go (Gin) pour le backend et React (Vite) pour le frontend. Elle utilise PostgreSQL pour la base de données et WebSocket pour la communication en temps réel.

L’application est responsive et adaptée aux écrans mobiles et desktop, avec un design moderne et attractif.

Fonctionnalités

Chat en temps réel avec WebSocket
Authentification utilisateur (JWT)
Historique des messages stocké dans PostgreSQL
Interface responsive pour mobile et desktop
Notifications visuelles pour les nouveaux messages
Déploiement séparé backend et frontend sur Render

Technologies

Backend : Go, Gin, Gorm, PostgreSQL, Gorilla WebSocket
Frontend : React, Vite, React Router, React Icons, CSS/SCSS
Base de données : PostgreSQL (Render)
Déploiement : Render (backend et frontend)
